# 엑스퍼트 HR 수강신청 Front

> HR교육 소개 및 수강신청 사이트

## 🔧 기술 스택

- svelte, vite
- package :
  - axios, svelte-spa-router, bootstrap
  - flatpickr, font-awesome, fullcalendar, jquery, jspreadsheet-ce, moment, summernote

## 기본 세팅

- nodejs 설치
- 패키지 파일들 설치
  - cd ./프로젝트명
  - npm install

## 실행 방법

- npm run dev  
- npm run dev -- --port 5173 => port5173로 로컬실행
> API쪽 CORS 확인필요 => localhost:5173 열려있음

## 빌드

- npm run build
